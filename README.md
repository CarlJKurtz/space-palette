# Space Palette

**Space Palette is a Figma plugin that gives you a one-click palette of typographic space characters.**

Click a character to copy it to your clipboard, then paste with `Cmd+V` / `Ctrl+V`.

## Characters

| Name | Unicode |
|------|---------|
| Hair Space | `U+200A` |
| Thin Space | `U+2009` |
| Six-per-em Space | `U+2006` |
| Four-per-em Space | `U+2005` |
| Three-per-em Space | `U+2004` |
| En Space | `U+2002` |
| Em Space | `U+2003` |
| Figure Space | `U+2007` |

## Installation

1. Download the [latest release](https://github.com/CarlJKurtz/space-palette/releases).
2. Unzip and open Figma Desktop.
3. Go to **Plugins → Development → Import plugin from manifest…**
4. Select `manifest.json` from the unzipped folder.
